# Quick Start Guide

Get up and running with the Gearbox Fault Detection System in minutes!

## Prerequisites

- Python 3.8 or higher
- pip package manager
- Web browser

## Installation (5 minutes)

### Option 1: Automated Setup (Recommended)

**Linux/macOS:**
```bash
chmod +x setup.sh
./setup.sh
```

**Windows:**
```bash
setup.bat
```

### Option 2: Manual Setup

```bash
# 1. Create virtual environment
python -m venv venv

# 2. Activate virtual environment
# Linux/macOS:
source venv/bin/activate
# Windows:
venv\Scripts\activate

# 3. Install dependencies
pip install -r requirements.txt

# 4. Generate sample data (optional)
python generate_sample_data.py
```

## Running the Application

```bash
# Start the server
python app.py

# The application will be available at:
# http://localhost:5000
```

## Your First Analysis (2 minutes)

1. **Open your browser** to `http://localhost:5000`

2. **Upload a file**:
   - Click "Choose File" or drag & drop
   - Use a sample file from `data/samples/` (generated by setup)
   - Supported formats: CSV, TXT, Excel (.xlsx, .xls), MATLAB (.mat)

3. **Click "Analyze Vibration Data"**

4. **View Results**:
   - Fault score and damage level
   - Detected fault types
   - Time and frequency visualizations
   - Maintenance recommendations

## Sample Data Files

After running setup, you'll find sample data in `data/samples/`:

- `healthy_signal.csv` - Normal operation
- `bearing_fault_signal.csv` - Bearing defect
- `unbalance_fault_signal.csv` - Rotor unbalance
- `misalignment_fault_signal.csv` - Shaft misalignment
- `gear_fault_signal.csv` - Gear mesh fault

Try analyzing each one to see different fault patterns!

## Understanding the Results

### Damage Levels

- **Healthy** (0-20%): ‚úÖ System operating normally
- **Slight** (20-40%): ‚ö†Ô∏è Minor abnormalities detected
- **Moderate** (40-60%): ‚ö†Ô∏è Moderate wear, inspection needed
- **Severe** (60-80%): üî¥ Severe damage, immediate attention required
- **Critical** (80-100%): üî¥ Critical condition, shutdown recommended

### Fault Types

- **Bearing Fault**: Bearing defects or wear
- **Unbalance**: Rotor mass imbalance
- **Misalignment**: Shaft misalignment
- **Gear Fault**: Gear tooth damage or wear
- **General Abnormality**: Other vibration anomalies

### Visualizations

1. **Time Domain Plot**: Raw vibration signal over time
2. **Frequency Domain (FFT)**: Frequency content analysis
3. **Spectrogram**: Time-frequency representation
4. **Feature Analysis**: Statistical feature comparison

## Troubleshooting

### Import Errors
```bash
# Ensure you're in the virtual environment
source venv/bin/activate  # Linux/macOS
venv\Scripts\activate     # Windows

# Reinstall dependencies
pip install -r requirements.txt
```

### Port Already in Use
```bash
# Use a different port
FLASK_PORT=8080 python app.py
```

Or modify `app.py` line 97 to change the port.

### File Upload Fails
- Check file size (max 16MB by default)
- Verify file format is supported
- Ensure file contains numeric data

### No Visualizations Shown
- Check browser console for errors
- Ensure matplotlib is installed correctly
- Try a different browser
